Приложение "Книги".
Функционал приложения:
    1. Добавление новой книги в коллекцию.
     2. Удаление книги из колекции.
     3. Поиск книги по названию.
     4. Сортировка колекции по заданному атрибуту.
     5. Вывод коллекцию.
     6. Чтение или запись коллекции в файл.
     7. Выход.
Закомментили где и какие строки за что отвечают.
Богдан занимался выполнением 1,2,3 функций.
Карим занимался выполнением 4,5,6.
